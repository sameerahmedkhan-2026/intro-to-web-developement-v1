# What is code?

Code is for humans first and computers second.
because you or someone else will have to go back and re-read that code some time in the future, maybe tomorrow or maybe in ten years.
it's important to write code in such a way that it can be understood as quickly as is reasonable.

- use meaningful variable and function names.
- use comments to explain why you are doing something, not what you are doing.
- use consistent formatting and indentation.
- avoid unnecessary complexity and abstraction.

- Code is communication.

- With JavaScript, only one thing is ever happening at a time (this is called being single threaded) In general, this means it executes line 1, then line 2, then line 3, etc.
