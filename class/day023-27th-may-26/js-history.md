# JavaScript

- created by Brendan Eich in 1995 working at Netscape within 10 days.
- initially named Mocha, then LiveScript, and finally JavaScript.
- designed to run in web browsers, enabling interactive web pages.
- standardized as ECMAScript by ECMA International.
- has evolved significantly, with major versions introducing new features and improvements.
- widely used for both client-side and server-side development, with frameworks like React, Angular, and Node.js.
