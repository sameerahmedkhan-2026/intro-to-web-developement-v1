CSS Specificity: What "0 0 0 1" Means

CSS specificity is represented as four numbers: (inline, IDs, classes/attributes/pseudo-classes, elements/pseudo-elements).

  • "0 0 0 1" means: 0 inline styles, 0 ID selectors, 0 class/attribute/pseudo-class selectors, and 1 element selector — for example, p or h1.
  • Examples:
    • h1 {} → specificity 0 0 0 1
    • div p {} → specificity 0 0 0 2 (two element selectors)
    • .card p {} → specificity 0 0 1 1 (one class and one element)
  • Comparison rule: compare the four numbers left-to-right; the larger number at the first difference wins.
  • !important overrides normal specificity. If multiple !important rules conflict, specificity is compared among them.
